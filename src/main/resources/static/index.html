SIHRA - Aplicación desplegada

Redirigiendo a /api. Si prefieres que la raíz (/) la atienda un controlador, añade un mapeo "/" en uno de tus controladores existentes.

Recursos útiles:

  • Ver endpoints (si tienes un controlador en /api)
  • Health (si Actuator está habilitado)
  • Swagger UI (si lo configuraste)

Si prefieres que lo atienda un controlador, pega el snippet Java que te di en una clase @RestController existente.
